## 🚀 Featured Projects

### 🤖 Sofia — 3D Intelligent Virtual AI Agent
> Multi-agent AI assistant with AR presence, real-time lip sync, and live computer vision personalisation.
> Analyses user **gender · age · emotion** in real time to deliver a personalised experience.
> Built as a smart customer assistant with clustering, a manager analytics dashboard, and persistent memory.

`Google ADK` `Gemini API` `FastAPI` `React` `TypeScript` `Unity AR` `MongoDB` `SQLite` `Microsoft Azure STT/TTS` `Hugging Face`

---

### 🎮 MotionX — Motion & Voice Controlled Game Series
> Three original games where **your body and voice are the controller** — no keyboard, no mouse.

| Game | Mechanic |
|---|---|
| **LumRun** | Run through a cemetery — controlled by running in place |
| **Music Bubble Runner** | Pop mid-air musical bubbles with your movement — creates a live soundtrack |
| **CWL (Charms & Wands League)** | Harry Potter-inspired — wand tracks your hand, voice detects spells like *Expelliarmus* |

`Computer Vision` `Speech Detection` `Python` `Real-time Tracking`

---

### 🛒 Groceria — Multi-Agent Smart Shopping Planner
> Cooperative multi-agent system that scrapes vendor sites in real time, reasons over prices, delivery costs,
> credit card discounts, and user preferences to output the most optimised shopping plan.

`Google ADK` `FastAPI` `Next.js` `Real-time Scraper` `Multi-Agent Reasoning`

🏆 *Finalist — SLAIC AI Challenge 2025*

---

### 📄 QuickRef — AI Document QA System
> End-to-end RAG pipeline: upload a PDF or URL → ask questions → get cited answers.
> Built from scratch with chunking, FAISS vector search, and LLM-powered responses.

`LangChain` `FAISS` `NVIDIA BGE-M3` `Falcon-7B` `Streamlit` `Python`

---

### 💬 AnoNote — Anonymous Messaging + NLP Harm Detection
> Anonymous messaging platform with a multilingual harm detection pipeline covering **English, Sinhala & Singlish**.
> Four custom TF-IDF models + SpaCy pipeline, all deployed on Hugging Face.

`MERN Stack` `FastAPI` `TF-IDF` `SpaCy` `Hugging Face` `NLP`
